# Price_Sensitivity_Analysis

## Project Overview
Food delivery price sensitivity analysis using Python and Tableau.

## Problem Statement
Analyze how price changes affect customer demand across 
meal categories and fulfilment centers.

## Tools Used
- Python (data cleaning & analysis)
- Tableau (visualization & dashboard)

## Dataset
- 77 fulfilment centers
- 51 meal items across 14 categories
- 4 cuisines: Thai, Indian, Italian, Continental

## Key Insights
- Avg Checkout Price: ₹327
- Total Orders: 1.35M
- Total Revenue: ₹359.57M
- Avg Discount: 6.44%
- Beverages is the top revenue category
